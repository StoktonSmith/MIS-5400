NHL Data

By Stokton Smith & Alex White

This data is was pulled using \'https://statsapi.web.nhl.com/api/v1/teams\'

To pull data please add these to the end of the url:

  • /NHL_data (Returns all data)
  • /NHL_teams (Returns all team data)
    • /NHL_teams/[Team ID*] (Returns specific team data)
  • /NHL_players (Returns all player data)
    • /NHL_players/[Player ID] (Returns specific player data)

For POST requests to NHL_Teams, here is an example JSON format:

[ { "City": "[City]", "Conference": "[Conference]", "Division": "[Division]", "Short_Name": "[Abbreviation]", "Start_Year": "[Start Year]", "State": "[State]", "Team_ID": [Team ID], "Team_Name": "[Full Name]", "Venue": "[Venue]", "Website_URL": "[URL]" } ]

For POST requests to NHL_PLAYERS, here is an example JSON format:

[ { "Jersey_Number": "[Jersey #]", "Player_ID": [Player ID], "Player_Name": "[Full Name]", "Position_Name": "[Position Name]", "Position_Type": "[Position Type]", "Team_ID": [Team ID] } ]


*If it is bolded, you replace it with whatever value you'd like to search/add
